Installation
------------

Install the ``alliance_platform_storage`` package:

.. code-block:: bash

    poetry add alliance_platform_storage

Add ``alliance_platform.storage`` to your ``INSTALLED_APPS``.

.. code-block:: python

    INSTALLED_APPS = [
        ...
        'alliance_platform.storage',
        ...
    ]
