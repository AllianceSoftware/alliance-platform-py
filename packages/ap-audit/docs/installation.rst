Installation
------------
Install the ``alliance_platform_audit`` package:
.. code-block:: bash
    poetry add alliance_platform_audit
Add ``alliance_platform.audit`` to your ``INSTALLED_APPS``.
.. code-block:: python
    INSTALLED_APPS = [
        ...
        'alliance_platform.audit',
        ...
    ]
